## <a name="pkg-overview">Overview</a>
Package cmdspliter intends to immitate bash way to split a string command to
an array of arguments

For example:
"/bin/cmd -f 'a.zip b.zip' -g" => ["/bin/cmd", "-f", "'a.zip b.zip'", "-g"]
